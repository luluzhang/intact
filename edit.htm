Scan&Solve™ Documentation

Restraints

Create and Edit Restraints

Scan&Solve™ can constrain selected faces in a model. A face can be restrained in all three coordinate directions, making it completely rigid, or a combination thereof. Any part of the model that is restrained will not be affected by deflection later on.

Creating Restraints

To create a restraint, press the [Add] (restraint) button in the Specify tab.

Editing Restraints

A restraint is annotated with a conical shape oriented in the direction of the restraint, and colored accordingly. By default, a restraint is applied in all three coordinate directions when first created. To partially restrain, press the [Edit] (restraint) button and select which directions to restrain.
