Installation

Run Scan&Solve_RhinoSetup.exe to install the Scan&Solve™ plugin. This will install all necessary Intel support libraries, as well as the Scan&Solve™ plugin.

If you have previously installed an older version of Scan&Solve™, use Windows control panel to un-install it before running Scan&Solve_RhinoSetup.exe.
